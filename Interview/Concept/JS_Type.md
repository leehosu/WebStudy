# 타입

## 자바스크립트 타입

- `undefined`, `null`, `boolean`, `number`, `string`, `object`, `function`, `array`, `Date`등

## 기본형과 참조형

##### 기본형

- `number`
- `string`
- `boolean`
- `Null`
- `undefined`

##### 참조형

- `Object`
  - `Array`
  - `Function`
  - `RegExp`

## 기본형과 참조형의 차이점

- 기본형 : `값`을 그대로 할당
- 참조형 : 값이 저장된 `주소`를 할당(참조)
