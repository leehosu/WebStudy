## 🎗 JavaScript란?
동적으로 컨텐츠를 바꾸고, 멀티미디어를 다루고, 움직이는 이미지등 웹 페이지를 꾸며주도록 하는 프로그래밍 언어이다.

## 👓 JavaScript 기능?
1. 변수안에 값을 저장 할 수 있다.
2. '문자열'이라고 불리는 문자들도 저장이 가능하다.
3. 웹 페이지 상의 이벤트에 따라 응답하는 코드를 운용할 수 있다.
4. block-level의 범위가 아닌 Fuction-level의 범위를 가진다.
